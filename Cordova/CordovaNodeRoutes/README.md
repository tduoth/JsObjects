# CordovaNodeRoutes

- Make sure the IP address is correct in **Control.js**. It should 
point at your running instance of **NodeRoutesParams**.
- Run: **cordova platform add android**
- Navigate to the **www** folder and run **bower install**.
- Run **./install **
